# React Shop

## Tech stack ⚡
- React
- Axios
- SWR
- Express
- Sqlite
- Vite

## Install 

```bash
git clone https://github.com/hb256/Shop-React.git .
pnpm install 
```
### Run React app
```bash
pnpm dev
```
### Run Server
```bash
pnpm sv
```
